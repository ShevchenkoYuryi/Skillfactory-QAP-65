d: - переход на диск D:

CD - сменить директорию

git clone <URL рипозитория> -  клонировать репозиторий с GitHub

git status - это основная команда для отслеживания изменений файлов, находящихся в репозитории

dir - выводит содержимое каталога

git add . - чтобы файл попал под версионный контроль и впоследствии в нём можно было зафиксировать изменения

git commit -m "..." - фиксация изменений с коммитами

git push - публикация изменений на сервере

git log - показывает историю изменений

git branch ... - создание новой ветки

git branch --list - выведет список веток которые существуют в репазитории

git branch --list -v - выведет инфу о последних коммитах каждой ветки

git checkout "имя ветки" - преход на ветку

git merge имя ветки - слияние веток

git pull - получаем изменения
